# 🚀 快速启动指南

## 📋 前置条件

确保你已经安装了：
- Node.js 18.0+ 
- npm 或 yarn

## ⚡ 5分钟快速体验

### 1. 启动项目
```bash
cd scientific-knowledge-navigation
npm install
npm run dev
```

### 2. 访问系统
打开浏览器访问：http://localhost:3000

### 3. 测试AI模型
1. 点击导航栏的 "Model Test" 或直接访问 http://localhost:3000/model-test
2. 在文本框中输入任何测试文本，例如："你好，你是谁？"或"中国有多少直辖市？"
3. 选择流式或非流式模式
4. 点击发送按钮
5. 观察AI模型的实时响应

### 4. 配置真实AI模型（可选）
1. 点击右上角的 "Configure" 按钮
2. 编辑模型配置，填入真实的API密钥
3. 保存配置后重新测试
4. 对比模拟API和真实API的响应差异

## 🧪 自动化测试

## 🔧 常见问题

### Q: Model Test Console的用途是什么？
A: 这是用于测试AI模型输入输出的工具，不是专门生成知识网络的。用户可以输入任何文本来测试模型的响应。

### Q: 如何使用真实的AI模型？
A: 在模型测试页面点击"Configure"，填入你的API密钥和端点即可。

### Q: 为什么使用模拟API？
A: 模拟API用于演示和测试，无需真实API密钥即可体验流式响应效果。

### Q: 流式和非流式有什么区别？
A: 流式模式会逐字符显示AI响应，无需等待完整结果；非流式模式会等待完整响应后一次性显示。

### Q: 如何查看系统日志？
A: 在模型测试页面切换到"System Logs"标签查看详细日志。
